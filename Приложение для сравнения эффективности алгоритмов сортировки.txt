Реализовать приложение на C# с использованием WinForms, поддерживающее возможность подключения плагинов.

Приложение позволяет сравнить эффективность различных алгоритмов сортировки массивов целочисленных значений.

Графический интерфейс пользователя представляет собой окно, состоящее из следующих основных элементов:
- список (возможно, cheboxlist), в котором перечислены названия подгруженных реализаций (например "Быстрая сортировка", "Сортировка вставками" и т.д.);
- текстовое поле для ввода размера входного набора чисел;
- текстовое поле для ввода количества прогонов сортировки;
- кнопка для запуска сортировки;
- панель для просмотра результатов работы (вид - произвольный) для каждого алгоритма:
	- название алгоритма;
	- минимальное время выполнения;
	- максимальное время выполнения сортировки;
	- среднее время выполнения сортировки.

Реализации алгоритмов сортировки находятся в отдельных библиотеках, физически расположенных в одной папке с основным исполняемым файлом.
При загрузке приложение производит поиск плагинов с реализациями алгоритмов и загружает их.
Пользователь должен иметь возможность написания своего плагина (динимачески подключаемой библиотеки) с собственной реализаций алгортима сортировки (к приложению должен быть приложен файл с текстовым описанием интерфейса, который должен реализовать пользователь). 

Желательно, чтобы в списке была возможность выбрать алгоритмы для сравнения.

Сценарий работы с приложением:
1. Пользователь вводит размер входного массива чисел
2. Пользователь вводит количество прогонов.
3. Пользователь выбирает интересующие его алгоритмы в списке.
4. Пользователь наживает кнопку, стартующую выполнение выбранных алгоритмов сортировки.

По нажатию кнопки последовательно выполняется количество итераций, равное количеству прогонов, введенному пользователю.
В каждой итерации генерируется массив с размером, указанным пользователем, и последовательно запускаются алгоритмы сортировки, принимающие на вход массив. 
По завершению обработки результаты выводятся в панели просмотра результатов.

Реализовать 2 плагина, реализующие быструю сортировку и пузырьковую сортировку.

Необязательно, но будет плюсом:
	- выполнить алгоритмы в отдельном потоке, чтобы UI не блокировался.
	- реализовать алгоритм, выполняющий сортировку с использованием нескольких потоков.
